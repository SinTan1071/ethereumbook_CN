# 精通以太坊

精通Ethereum是针对开发人员的一本书，提供了操作和使用Ethereum、Ethereum Classic、RootStock（RSK）以及其他基于EVM（以太坊虚拟机）的开放区块链的指南。

这个资料库包含这本书的初稿，目前正在进行中，计划由O' reilly Media在2018年第四季度出版。不幸的是，由于一些个人问题，出版日期被推迟了。预计将在2018年第一季度和第二季度看到开发和推出的内容。
